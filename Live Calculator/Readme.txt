The aim of this project is to design a visual Calculator. This is a calculator that takes in visual feed and solves it and then returns output on the screen

	1) Detect Each symbol (numbers + operator)
	2) Understand how number order works : Unit, Hundred and Thousands
	3) Solve with operator used
	4) Return a perfect output using OpenCV that includes a bounding box and characters added
